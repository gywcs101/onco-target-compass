# 靶点卡片字段设计

## 1. 基础标识

靶点标准名、别名、基因/蛋白 ID、突变或构象、分析适应症、癌种、治疗线次、患者亚群、药物形式、卡片版本和更新时间。

## 2. 生物学与患者

机制摘要、通路关系、因果证据、人类遗传学证据、肿瘤表达、正常组织表达、表达异质性、突变/拷贝数/融合、候选生物标志物和检测方法。

## 3. 成药与临床

可成药位点、药物形式、暴露与组织分布、代表药物、试验编号、阶段、入组条件、剂量、主要终点、疗效、安全性、耐药和联合策略。

## 4. 竞争与决策

竞品项目、公司、临床线次、专利线索、标准治疗、未满足需求、失败事件、反向证据、证据缺口、差异化假设、验证问题和专家意见。

## 5. 每条结论的最小证据结构

`claim_id`、结论文本、事实/推断/建议类型、来源类型、来源 ID、原文片段、定位信息、研究条件、适用人群、发布时间、抓取时间、支持/反驳属性、证据等级、模型置信标记、人工复核状态和版本号。
